# Projeto de Automação de Cadastro em ERP

## Automação de Cadastro
Projeto para automatizar o cadastro de produtos em um sistema ERP utilizando Python e PyAutoGUI.
### Descrição
Este projeto tem como objetivo facilitar o processo de cadastro de produtos em um sistema ERP, automatizando tarefas repetitivas através de scripts em Python.
### Estrutura do Projeto
- `main.py`: Arquivo principal que contém o script de automação.
- `README.md`: Documentação do projeto.
- `.gitignore`: Arquivo para especificar quais arquivos ou pastas o Git deve ignorar.
- `requirements.txt`: Lista de dependências necessárias para o projeto.
- `base_erp_completa.xlsx`: Planilha base utilizada para o cadastro dos produtos.
- `index.html`: Página inicial do projeto web.
- `gerar_excel.py`: Script para gerar arquivos Excel a partir de dados fornecidos.
### Tecnologias Utilizadas
- Python
- PyAutoGUI
- OpenPyXL
### Instruções de Uso
1. Clone o repositório.
2. Instale as dependências listadas em `requirements.txt`.
3. Execute o script `main.py` para iniciar a automação do cadastro.
